Pila®

Gerenciador Financeiro

Pila é um gerenciador financeiro simples para controle de despesas pessoais. O aplicativo permite armazenar informações sobre as despesas, tais como data, valor, quantidade e tags. As tags são palavras-chave utilizadas para indicar a natureza dos gastos. Tags utilizadas uma vez são armazenadas e fornecidas como sugestão em cadastros subsequentes.

O aplicativo possui código aberto sob a Licença Apache e foi desenvolvido pela turma do curso de extensão Desenvolvimento Android, oferecido pela FTEC Faculdades em outubro de 2012. Segue uma lista com os autores originais:
